# Tematica - Sitio estatico

Proyecto estatico listo para desplegar en Vercel.

## Estructura

- `index.html`
- `styles.css`
- `script.js`
- `fotos/`
- `vercel.json`

## Deploy en Vercel

1. Sube esta carpeta a un repositorio (GitHub/GitLab/Bitbucket).
2. En Vercel, crea un proyecto nuevo desde ese repositorio.
3. Framework Preset: `Other`.
4. Build Command: vacio.
5. Output Directory: vacio (raiz del proyecto).
6. Deploy.

Tambien puedes desplegar por CLI:

```bash
vercel
vercel --prod
```
